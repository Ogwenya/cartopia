# Cartopia API

The documentation for Cartopia API

## Installation

```bash
$ npm install
```

## Env Variables

##### Database connection

- `DATABASE_URL`

##### Authentication

- `JWT_SECRET`

##### Password reset

- `FRONTEND_URL`

##### Email connection

- `EMAIL_ADDRESS`
- `EMAIL_PASSWORD`
- `EMAIL_HOST`
- `EMAIL_PORT`

##### Cloudinary Media Management

- `CLOUDINARY_NAME`
- `CLOUDINARY_API_KEY`
- `CLOUDINARY_API_SECRET`

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
